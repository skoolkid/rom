2AEE: THE 'DE,(DE+1)' SUBROUTINE
Used by the routine at STK_VAR.
This subroutine performs the construction 'LD DE,(DE+1)' and returns HL pointing to 'DE+2'.
Input
DE Target address minus one
Output
DE Word at the target address
HL Target address plus one
DE_DE_1 2AEE EX DE,HL Use HL for the construction.
2AEF INC HL Point to 'DE+1'.
2AF0 LD E,(HL) In effect - LD E,(DE+1).
2AF1 INC HL Point to 'DE+2'.
2AF2 LD D,(HL) In effect - LD D,(DE+2).
2AF3 RET Finished.
