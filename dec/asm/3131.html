03131: THE 'PO-SAVE' SUBROUTINE
Used by the routines at PO_TV_2 and PO_MSG.
The routine at PO_MSG continues here.
This subroutine allows for characters to be printed 'recursively'. The appropriate registers are saved whilst PRINT_A_1 is called.
Input
A Character code
PO_SAVE 03131 PUSH DE Save the DE register pair.
03132 EXX Save HL and BC.
03133 RST 16 Print the single character.
03134 EXX Restore HL and BC.
03135 POP DE Restore DE.
03136 RET Finished.
