05698: THE 'CHANNEL 'S' FLAG' SUBROUTINE
The address of this routine is derived from an offset found in the channel code lookup table.
Input
Output
CHAN_S 05698 RES 0,(IY+2) Signal 'using main screen' (reset bit 0 of TV-FLAG).
This entry point is used by the routine at CHAN_K.
CHAN_S_1 05702 RES 1,(IY+1) Signal 'printer not being used' (reset bit 1 of FLAGS).
05706 JP TEMPS Exit via TEMPS so as to set the colour system variables.
