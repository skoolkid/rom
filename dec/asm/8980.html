08980: THE 'STK-TO-A' SUBROUTINE
Used by the routine at STK_TO_BC.
This subroutine loads the A register with the floating point number held at the top of the calculator stack. The number must be in the range 0 to 255.
Output
A Number from the calculator stack
C Sign of the number (1 or 255)
STK_TO_A 08980 CALL FP_TO_A Modulus of rounded last value to A if possible; else, report error.
08983 JP C,REPORT_B_3
08986 LD C,1 One to C for positive last value.
08988 RET Z Return if value was positive.
08989 LD C,255 Else change C to 255 (i.e. minus one).
08991 RET Finished.
