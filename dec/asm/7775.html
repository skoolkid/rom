07775: THE 'CONTINUE' COMMAND ROUTINE
The address of this routine is found in the parameter table.
The required line number and statement number within that line are made the object of a jump.
CONTINUE 07775 LD HL,(23662) The line number (OLDPPC).
07778 LD D,(IY+54) The statement number (OSPCC).
07781 JR GO_TO_2 Jump forward.
