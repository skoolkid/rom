23665: FLAGX - Various flags
Read by the routines at EDITOR, ED_EDIT, ED_DOWN, ED_UP, ED_SYMBOL, SET_HL, OUT_LINE, LN_FETCH, OUT_SP_2, NEXT, IN_ASSIGN and LET, and updated by the routines at MAIN_EXEC, VAR_A_1 and INPUT. This system variable is accessed by most routines via the IY register at IY+55.
Bit Description
0 Set when dealing with a complete simple string
1 Set when dealing with a new (rather than existing) variable
2 Unused
3 Unused
4 Unused
5 Reset when in editing mode, set when in INPUT mode
6 Unused
7 Set when dealing with INPUT LINE
